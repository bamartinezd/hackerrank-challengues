# HackerRank Challenges - .NET Core Edition

Este repositorio contiene una colección de soluciones a desafíos de algoritmos y lógica de HackerRank, junto con pruebas técnicas de plataformas como Codility e Intertec.

## 📱 Entorno de Desarrollo Actual
A partir de 2026, este repositorio se mantiene y ejecuta en un entorno de movilidad avanzada:
- **Dispositivo:** Samsung Galaxy S25 Ultra (SM-S938B)
- **S.O:** Android 16 vía Samsung DeX
- **Kernel:** 6.6.77-android15 (aarch64)
- **Terminal:** Termux + Ubuntu Proot
- **IDE:** VS Code (code-server) ejecutándose localmente

## 📂 Estructura de la Solución
La solución principal `HackerRank-Challenges.sln` agrupa los proyectos por categorías de dificultad y tipo de estructura de datos.

### Desafíos Destacados
*   **Manipulación de Arreglos:** `ArrayManipulation`, `LeftRotate`, `DynamicArray`.
*   **Matrices:** `DiagonalMatrix`.
*   **Algoritmos de Búsqueda y Ordenamiento:** `Quicksort1Partition`, `BruteForceAlgorithm`.
*   **Estructuras de Datos Avanzadas:** `balanced-brackets`, `contacts`, `find-the-running-median`, `height-of-a-binary-tree`.
*   **Pruebas Técnicas Corporativas:**
    *   `AmazonTransactionLogs` / `AmazonDeliveryCenters`
    *   `TiendaPagoCodilityTest`
    *   `WordParserIntertec`
